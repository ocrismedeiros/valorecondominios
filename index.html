⚙️ Setup — Criar Tabela no Supabase

Execute o SQL abaixo no SQL Editor do seu projeto Supabase para criar a tabela de controle de tarefas.

1
Acesse supabase.com → seu projeto → SQL Editor → New query
2
Cole e execute o SQL abaixo clicando em RUN:
CREATE TABLE IF NOT EXISTS task_completions ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, collaborator text NOT NULL, task_key text NOT NULL, month_year text NOT NULL, done boolean DEFAULT false, completed_at timestamptz, updated_at timestamptz DEFAULT now(), UNIQUE(collaborator, task_key, month_year) ); ALTER TABLE task_completions ENABLE ROW LEVEL SECURITY; CREATE POLICY "public_access" ON task_completions FOR ALL TO anon USING (true) WITH CHECK (true); ALTER publication supabase_realtime ADD TABLE task_completions;
3
Aguarde "Success" e clique abaixo:
